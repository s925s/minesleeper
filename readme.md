# LuckSweeper

Python (PySide6) で動作する、高機能マインスイーパ・シミュレーター。
独自の地形生成アルゴリズム「Sea Mode」および、決定論的ロジックに基づく自動攻略ボットを搭載する。

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![PySide6](https://img.shields.io/badge/GUI-PySide6-green.svg)

## 概要 (Overview)

本ソフトウェアは、単なるマインスイーパではない。
「運」と「論理」の境界線を可視化し、プレイヤーの思考を補助、あるいは代行するツールである。

### 特徴

* **Sea Mode (海モード)**:
    * ランダム配置の常識を覆す、群島生成アルゴリズムを実装。
    * 「海」「砂浜」「陸」の概念を取り入れ、地形読みという新たな戦略性を提示する。
* **Logic Bot (論理演算ボット)**:
    * AI（機械学習）のような不確定なブラックボックスではない。
    * 純粋な数学的ロジックに基づき、盤面を走査・解析する自動化プログラム。
    * 「島攻略（角優先）」等の戦略プロファイルを切り替え可能。
* **UI/UX**:
    * GTAライクな "WASTED" / "MISSION PASSED" オーバーレイ演出。
    * Modern / Sea / Classic の3種のテーマ。
    * 盤面サイズ、爆弾密度、演算速度の完全なパラメータ制御。

## 実行方法 (Usage)

`main.py` を実行せよ。

※ 同階層に `win.wav`, `lose.wav` を配置することで、サウンド機能が有効化される。
